# Simple Inventory Control App
A simple app to register your inventory for Windows, Android and iOS. I use <a href="https://dotnet.microsoft.com/en-us/apps/maui">.NET MAUI </a>

## Minimum Requirements

- Android 6.0<br>
- Windows 10<br>
- iOS 14.2<br>
